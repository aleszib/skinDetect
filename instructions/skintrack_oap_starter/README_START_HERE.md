# SkinTrack OAP Starter Package

This package contains starter governance and planning files for the SkinTrack Change Monitor project.

Recommended initial placement in repository root:

```text
AGENTS.md
docs/PROJECT_MEMORY.md
docs/ARCHITECTURE.md
docs/MILESTONES.md
docs/DECISION_LOG.md
prompts/PR-001-codex-work-order.md
```

Suggested first action:

1. Copy these files into a new repository.
2. Review `AGENTS.md` and edit anything project-specific.
3. Start Codex CLI from the hardened WSL Ubuntu repository root.
4. Give Codex the contents of `prompts/PR-001-codex-work-order.md`.
5. Require a branch, tests, and final report.

The first coding PR should be backend-only and include tests.
